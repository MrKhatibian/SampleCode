Convert Feature to Raster

Цей інструмент перетворює об'єкти в набори растрових даних.

Будь-який клас об’єктів, що містить точкові, лінійні або полігональні об’єкти, може бути перетворений в набір растрових даних.

The cell center is used to decide the value of the output raster pixel.

The input field type determines the type of output raster. If the field is integer, the output raster will be integer; if it is floating point, the output will be floating point.

Choose feature layer to convert

The input feature layer to convert to a raster dataset.

Choose field

Choose the field that will be used to assign values to the output raster.

Output cell size

Розмір осередку і одиниці виміру для вихідних растрів.

The available units are Miles, Feet, Kilometers, and Meters. The default is Feet.

Result layer name

Назва шару, який буде створено в Мої ресурси і додано до карти. Назва за замовчуванням базується на назві інструменту та назві вхідного шару. Якщо шар уже існує, вам буде запропоновано вказати іншу назву.

За допомогою розкривного списку Зберегти результат в можна вказати назву папки в Мої ресурси, куди буде збережено результат.
