Extract Raster

Цей інструмент вирізає растр по кордону, заданому або як прямокутна область, або у відповідності з фігурою, обведеною на екрані. Ви можете вирізати растр за поточним екстентом області, який відображається на карті, або за областю вивчення, заданою полігоном.

Choose input data to extract

The input layer to clip.

Specify clipping geometry

Choose the extent type to use when clipping the geometry of the layer:

  • Use current extent—The current map extent will be used to clip out the resulting layer. This is the default.
  • Choose study area—Allows you to choose whether the result will show the area within the geometry extent or the area outside the geometry extent. If you choose the Data inside the geometry option, the imagery outside the extents will be removed. If you choose the Data outside the geometry option, the imagery within the extents will be removed. You can also define a clipping geometry using the draw tool.

Result layer name

Назва шару, який буде створено в Мої ресурси і додано до карти. Назва за замовчуванням базується на назві інструменту та назві вхідного шару. Якщо шар уже існує, вам буде запропоновано вказати іншу назву.

За допомогою розкривного списку Зберегти результат в можна вказати назву папки в Мої ресурси, куди буде збережено результат.
